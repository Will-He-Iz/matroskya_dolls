Relative Positioning

Relative positioning keeps the element in its original position. The space it would normally be is still occupied. However, it gives you the abiilty to use the positining commands to change where it will display. Note that the original space will still be occupied so elements around the object will not adjust.



div-1 { background-color: #cccccc; position: relative; top: 30px; left: -40px; }
